# GymLog — Kişisel Fitness Tracker

Haftalık antrenman, supplement, beslenme ve sağlık takip uygulaması.

## Özellikler
- 📊 Dashboard & BMI hesabı
- 🏋️ Antrenman & egzersiz takibi
- 💊 Supplement takibi
- 🍽️ Beslenme & makro takibi
- ❤️ Sağlık & recovery skoru
- 📈 Haftalık/aylık/yıllık grafikler
- 📲 Apple Health, Huawei, Samsung, Garmin import
- 🌐 Türkçe / English dil desteği
- 📱 PWA — telefona yüklenebilir

## Kurulum
1. Bu repo'yu fork et
2. Settings → Pages → Branch: main → / (root) → Save
3. `https://KULLANICI_ADI.github.io/gymlog` adresine git
4. Telefonda Safari/Chrome → "Ana Ekrana Ekle"

## Veri Güvenliği
Tüm veriler **tarayıcının localStorage**'ında saklanır.
Yedek için Settings → "JSON Dışa Aktar" kullan.
